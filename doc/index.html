VersionOne Search

Introduction

VersionOne Search (V1Search) is a standalone web application that allows individuals across your organization to search for VersionOne assets across all projects. V1Search is built on top of Lucene.NET, a powerful open-source information retrieval engine. This technology also supports a rich query parser that allows users to create queries that mix terms, logical expressions, and wildcards to narrow their searches.

System Requirements

  • Windows:
    • Tested with Windows 2000, 2003
  • Framework:
    • Microsoft .NET 4.0
  • Web Server:
    • IIS 5 or IIS 6, with ASP.Net 2.0 Service Pack 1
  • VersionOne:
    • 8.0 or above, including Team Edition

Download

The latest version of V1Search is available from the VersionOne Community site

Installation

  1. Determine Install Location

    V1Search can be installed on any web server that has internet access to your VersionOne instance. Exact placement should be determined by your internal software management requirements. The integation server must meet the System Requirements stated above.

  2. Install
    • Download

      Download the V1Search binary zip using the link above.

    • Extract to Web Server

      Extract the contents to a location on your web server.

    • Create Virtual Directory

      Create a virtual directory in IIS that points to the folder you extracted the files to.

    • Enable ASP.NET

      Ensure that the virtual directory is using ASP.Net 2.0 Service pack 1.

    • Default to index.htm

      Ensure that the default document is index.htm.

  3. Test the Web Application

    At this point you'll want to ensure that the Web Application is working. Open a web browser and navigate the the URL defined in the install step. You should see the VersionOne Search web page.

  4. Index your VersionOne Server

    To index your server navigate the the Extractor subdirectory. From this location, you need to modify the V1Extractor.exe. config file and provide your server connection information. The following table describes the parameters available to you in the configuration file.

    Parameter Description
    V1Server This is the URL to your VersionOne server
    V1User A valid VersionOne user in the system specifed by V1Server
    V1Password Password for the specified user
    IndexDir Location for storing Lucene index

    The index will only contain items the V1User has privileges to view. Therefore, if you use the admin user, then the index will contain everything. Conversely, if you choose a user that is restricted to a given project or has "Inheritor" access, you index may contain no data.
